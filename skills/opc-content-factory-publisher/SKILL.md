# 多平台发布包生成 + 封面图 + 同步发布

基于定稿自动生成 4 种平台适配版本，可选生成封面图，可选一键同步发布。

## 前置条件

- `04-定稿.md` 已完成（作者审阅修订后）
- 稿件文件夹结构完整

## 三阶段工作流

### 阶段一：生成多平台发布包（自动）

1. 在当前稿件文件夹下创建 `05-发布包/` 目录
2. 读取定稿内容
3. **扫描定稿中的图片**：
   - 检测定稿中所有 `![...](...)` 引用，记录每张图片的路径和上下文（图片前后的文字段落）
   - 如果定稿包含图片，根据各模板的图片处理规则分别处理
   - 如果定稿不含图片，正常生成，不添加图片相关内容
4. 分别使用四种适配模板生成发布包：

| 发布包 | 适用平台 | 使用模板 |
|--------|---------|---------|
| 长文发布包.md | 微信公众号、掘金、知乎 | `references/template-longform.md` |
| 图文发布包.md | 小红书 | `references/template-image-text.md` |
| 短视频口播文案发布包.md | 抖音、视频号 | `references/template-short-video.md` |
| 长视频口播文案发布包.md | B站、YouTube、西瓜视频 | `references/template-long-video.md` |

5. 每种适配需读取：
   - 定稿内容（`04-定稿.md`）
   - 写作风格分析报告（`00-模板库/作者个人写作风格分析报告.md`）
   - 账号基础信息（`00-模板库/自媒体账号基础信息.md`）

6. **字数限制（严格遵守）**：
   - **图文发布包（小红书）正文内容不得超过 1000 字**（不含标题）
   - 生成后必须统计正文字数，如超限则精简内容至 1000 字以内
   - 精简原则：保留核心观点和干货，删减次要论述和过渡段落，确保逻辑完整
   - 其他发布包无字数限制

7. 生成过程中如有不确定的信息，主动询问用户
8. **严格分离文件内容和建议信息**：
   - **写入文件的**：只有能直接复制到平台发布的内容（标题 + 正文/文案）
   - **在对话中说的**：封面图建议、配图建议、话题标签、发布时间建议、排版建议等
   - 发布包文件必须拿到就能直接发布，不需要手动删除任何内容
9. 完成后告知用户发布包文件位置，并在对话中给出建议性信息
10. **按需建议用户使用 dbskill 工具优化发布包**：
   - 短视频/长视频口播包：建议用 `/dbs-hook` 优化开头 3 秒钩子（10-15 种改写方案）
   - 小红书图文包：建议用 `/dbs-xhs-title` 优化标题（75 个爆款公式匹配）
   - 以上为可选步骤，不强制执行

### 阶段二：生成封面图（可选）

询问用户是否需要生成封面图。如果需要：

1. 读取 `references/cover-generator-guide.md` 获取封面图生成指南
2. 确认主标题和高亮关键词
3. 确认文案（英文标签、引导句、签名）
4. 推荐样式
5. 选择目标平台
6. 执行封面图生成命令

也可以让用户直接使用 `/cover-generator` 独立生成。

### 阶段三：多平台同步发布（可选）

询问用户是否需要同步发布。如果需要：

1. 读取 `references/wechatsync-guide.md` 获取同步发布指南
2. 确认 wechatsync 前置条件已满足
3. 检查平台登录状态：`WECHATSYNC_TOKEN="<your-wechatsync-token>" wechatsync platforms --auth`
4. 用户选择目标平台和对应发布包
5. 执行同步命令，默认为草稿模式
6. 报告发布结果和草稿 URL

也可以让用户直接使用 `/wechatsync` 独立发布。

### 阶段四：发布后复盘（可选·建议）

发布并获得数据反馈后，建议用户：

1. 用 `/dbs-content` 做内容五维诊断（写作干净度、封面/标题、表达效率、认知差、AI 工作流）
2. 用 `/dbs-decision` 把踩坑或爆款经验沉淀到个人决策知识库（`~/.dbs/decisions/`）
3. 用 `/dbs-slowisfast` 识别写作流程中哪些环节值得"慢做"沉淀资产

## 注意事项

- 四种适配版本的核心内容必须来自定稿，只做平台适配
- 封面图和同步发布为可选步骤，不强制执行
- 发布包默认以草稿模式同步，用户确认后才正式发布
- 如用户只需部分功能（如只要发布包、只要封面图），按需执行对应阶段
