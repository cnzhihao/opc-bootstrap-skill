# 高价值选题自动筛选

从日常素材中批量筛选高价值选题，自动生成标准化选题笔记存入选题库。

## 前置条件

- `03-素材库/` 中已积累足够的素材内容
- 已确认内容领域和目标受众画像（参考 `00-模板库/自媒体账号基础信息.md`）

## 工作流

### 1. 收集素材

从 `03-素材库/` 中按以下规则获取素材：

1. 扫描 `03-素材库/` 中所有 `.md` 文件（不包括 `01-历史文章素材/` 子文件夹）
2. 按 frontmatter 中的 `date` 字段从新到旧排序
3. AI HOT 日报类型的素材（`source: "AI HOT 日报"`），优先读取最近 3 份（信息密度最高）
4. 然后读取其他来源的素材（X 剪藏、微信剪藏等），同样从新到旧
5. 如果素材不足，提示用户先积累素材

### 2. 执行筛选

读取 `references/prompt-topic-picker.md` 中的完整 Prompt，按以下步骤执行：

1. 从素材中筛选不少于 5 个高价值选题
2. 每个选题包含：选题标题、目标受众、核心痛点、3 个切入角度、创作优先级、素材来源
3. **选题标题必须严格符合个人风格**：读取 `00-模板库/作者个人写作风格分析报告.md` 中的「标题风格」维度，标题口语化、有态度、有第一人称、有具体数字/事件，禁止论文腔/散文腔/标语腔/鸡汤腔
4. 使用 `00-模板库/模板-选题笔记.md` 模板，为每个选题生成标准化选题笔记
5. 按选题类型存入对应子文件夹：
   - 热点选题 → `01-选题库/01-热点选题池/`
   - 用户痛点选题 → `01-选题库/02-用户痛点选题池/`
   - 行业干货选题 → `01-选题库/03-行业干货选题池/`
6. 最后给出整体选题评估报告

### 3. 输出确认

- 告知用户生成了多少个选题
- 列出每个选题的标题和优先级
- 告知文件保存位置
- 请用户从生成的选题中选择一个进行创作

### 4. 用户选定选题后执行

用户选定一个选题后，自动执行以下操作：

#### 4a. 移动选中的选题笔记到稿件库

通过 bash 命令将选中的选题笔记移动到稿件文件夹中：

```bash
TITLE="实际的稿件标题"
DATE_SUFFIX=$(date +%Y%m%d)
FOLDER_NAME="${TITLE}-${DATE_SUFFIX}"
FOLDER_PATH="02-稿件库/01-创作中稿件/${FOLDER_NAME}"

# 创建稿件文件夹
mkdir -p "${FOLDER_PATH}/assets"

# 移动选题笔记到稿件文件夹
mv "01-选题库/XX-XXX选题池/选题笔记文件名.md" "${FOLDER_PATH}/00-选题笔记.md"
```

#### 4b. 清空选题池（保留个人感悟选题池）

删除除「个人感悟选题池」以外的所有选题池中的文件：

```bash
# 清空热点选题池、用户痛点选题池、行业干货选题池
rm -f 01-选题库/01-热点选题池/*.md
rm -f 01-选题库/02-用户痛点选题池/*.md
rm -f 01-选题库/03-行业干货选题池/*.md
# 注意：不清空 01-选题库/04-个人感悟选题池/

# 确保空文件夹仍然存在（rm 删光文件后 macOS 可能清理空目录）
mkdir -p 01-选题库/01-热点选题池 01-选题库/02-用户痛点选题池 01-选题库/03-行业干货选题池
```

#### 4c. 完成确认

- 告知用户选题笔记已移动到稿件文件夹
- 告知用户非感悟选题池已清空，保持选题池新鲜
- **建议用户在访谈前用 `/dbs-goal` 做一次选题目标审计**（可选但推荐）：确认这篇稿子到底要交付什么，防止自嗨选题浪费访谈时间
- 也可用 `/dbs-deconstruct` 拆解核心概念，确保选题观点不含糊
- 审计完成（或跳过）后，使用 `/content-interview` 启动访谈

## 筛选标准（必须严格遵守）

1. **强相关**：必须和内容领域强相关
2. **有痛点**：必须击中目标受众的真实痛点
3. **有价值**：能提供情绪/实用/认知价值
4. **有壁垒**：能结合个人经验输出独家观点
5. **可落地**：能拓展成完整自媒体内容

## 注意事项

- 禁止生成泛泛而谈、全网同质化的选题
- 所有选题必须基于提供的素材，有对应支撑
- 禁止生成不符合个人定位的选题



## Suite 合并关系

本 skill 是内容工厂的选题主入口，已吸收选题优先级评分能力。

处理选题时先完成选题池梳理，再按商业价值、表达确定性、热点窗口、客户转化、资产沉淀价值进行优先级判断。
