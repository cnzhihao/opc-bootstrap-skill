# 五轮优化 Prompt

适用技能：`opc-content-factory-draft-optimizer`

## 优化目标

将初稿逐轮优化为可发布文章，每轮只解决一个主要问题，保留可追踪版本差异。

## 五轮流程

### 第 1 轮：结构重排

检查主张、段落顺序、论证链路和信息密度，删掉无效铺垫。

### 第 2 轮：案例补强

补足具体场景、动作、冲突、结果和复盘，避免抽象判断悬空。

### 第 3 轮：语言去 AI 味

替换模板化表达、过度总结、空泛金句和论文腔，增强作者口吻。

### 第 4 轮：传播性优化

优化标题、开头、小标题、金句、转折和结尾，让读者更容易读完和转发。

### 第 5 轮：发布前校验

检查事实、敏感表达、承诺边界、渠道适配和格式完整性。

## 每轮输出

```markdown
## 本轮目标
## 修改摘要
## 修改后正文
## 仍需确认的问题
```

## 验收标准

- 每轮目标清晰，不混合所有问题一起改。
- 保留作者真实判断，不把内容改成通用模板。
- 最终稿可交给发布包或归档流程。
