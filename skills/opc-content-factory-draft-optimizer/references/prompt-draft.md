# 初稿生成 Prompt

适用技能：`opc-content-factory-draft-optimizer`

## 目标

基于访谈记录、选题笔记和作者风格，生成一版可继续优化的长文章初稿。

## 输入

- 选题与目标受众
- 访谈记录或素材笔记
- 核心观点与案例
- 作者风格要求
- 目标渠道与篇幅

## 生成要求

1. 先明确文章主张，不写泛泛导语。
2. 用真实场景、冲突和判断展开，不用模板化鸡汤。
3. 保留作者第一人称判断和口语感。
4. 每个大段只服务一个观点，避免堆概念。
5. 结尾给出行动建议或判断沉淀。

## 输出格式

```markdown
# 标题

## 开头

## 正文

## 结尾

## 可继续优化的问题
```

## 验收标准

- 有明确主张和目标受众。
- 至少包含 2 个具体案例或场景。
- 没有明显 AI 味套话。
- 可进入五轮优化流程。
