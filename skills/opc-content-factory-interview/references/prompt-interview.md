# 深度访谈 Prompt

适用技能：`opc-content-factory-interview`

## 目标

围绕已确定选题，挖掘创始人的真实经历、判断、案例和金句，形成可用于文章、视频或复盘的访谈记录。

## 执行流程

1. 先确认选题、目标受众、内容用途和预期发布渠道。
2. 将问题拆成背景、关键经历、判断依据、失败教训、方法沉淀、可公开金句六类。
3. 每次只问一个问题；用户回答后追问细节、时间、人物、冲突、结果和复盘。
4. 对抽象判断追问“当时发生了什么”“你怎么判断的”“如果重来会怎么做”。
5. 访谈结束后输出访谈记录、核心观点、文章骨架和标题候选。

## 输出格式

```markdown
## 访谈记录

### 选题

### 核心观点

### 关键经历

### 可用案例

### 金句候选

### 文章骨架

### 标题候选
```

## 验收标准

- 至少沉淀 3 个真实案例或具体场景。
- 至少产出 5 条可用金句。
- 文章骨架能直接交给初稿优化流程。
- 不用空泛总结替代用户真实表达。
